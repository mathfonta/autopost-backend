## Descrição

Como AutoPost, preciso que o Agente Designer processe a imagem conforme a decisão do Analista — foto de obra: adicionar logo discreto e salvar limpa; card clean: criar cartão com fundo sólido e texto — e fazer upload do resultado final para o Cloudflare R2.

---

## Critérios de Aceitação

- [x] AC1: Para `content_type=obra_realizada` e `publish_clean=True`: adiciona logo no canto inferior direito (transparência 60%), salva no R2
- [x] AC2: Para tipos card (`dica`, `promocao`): cria imagem 1080x1080px com fundo na cor primária do cliente, texto em destaque, logo
- [x] AC3: Para `content_type=antes_depois`: cria layout dividido (50%/50%) com labels "ANTES"/"DEPOIS" e logo
- [x] AC4: Imagem final salva no R2 com nova chave `processed/{request_id}/final.jpg`
- [x] AC5: `design_result` atualizado com `processed_photo_url` (URL pública do R2), `type`, `dimensions`
- [x] AC6: Imagem final ≤ 8MB (limite Meta Graph API)
- [x] AC7: Testes unitários com imagens de fixture (não chama R2 real)
