# Story 2.3 — Agente Designer (Pillow)

**Status:** Draft  
**Epic:** 2 — Agentes de IA  
**Story:** 2.3  
**Estimativa:** 3 pontos  
**Depende de:** 2.2

---

## Descrição

Como AutoPost, preciso que o Agente Designer processe a imagem conforme a decisão do Analista — foto de obra: adicionar logo discreto e salvar limpa; card clean: criar cartão com fundo sólido e texto — e fazer upload do resultado final para o Cloudflare R2.

---

## Critérios de Aceitação

- [ ] AC1: Para `content_type=obra_realizada` e `publish_clean=True`: adiciona logo no canto inferior direito (transparência 60%), salva no R2
- [ ] AC2: Para tipos card (`dica`, `promocao`): cria imagem 1080x1080px com fundo na cor primária do cliente, texto em destaque, logo
- [ ] AC3: Para `content_type=antes_depois`: cria layout dividido (50%/50%) com labels "ANTES"/"DEPOIS" e logo
- [ ] AC4: Imagem final salva no R2 com nova chave `processed/{request_id}/final.jpg`
- [ ] AC5: `design_result` atualizado com `processed_photo_url` (URL pública do R2), `type`, `dimensions`
- [ ] AC6: Imagem final ≤ 8MB (limite Meta Graph API)
- [ ] AC7: Testes unitários com imagens de fixture (não chama R2 real)

---

## Dev Notes

- Pillow já está no requirements.txt
- boto3 compatível com R2 (endpoint customizado em `CLOUDFLARE_R2_ENDPOINT`)
- Logo do cliente vem de `brand_profile.logo_url` ou logo padrão AutoPost
- `brand_profile.primary_color` deve ser hex string (ex: `#1A3C6E`)
- Redimensionar imagem para 1080x1080 antes de processar (Instagram square)
