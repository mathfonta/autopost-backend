## Descrição

Como AutoPost, preciso que o Agente Publicador publique posts aprovados no Instagram Business e Página do Facebook via Meta Graph API, registre os IDs de publicação e inicie a coleta de métricas 24h depois.

## Dev Notes

- Instagram: processo em 2 etapas — `POST /{ig-id}/media` → `POST /{ig-id}/media_publish`
- Erro code 190/102 = token expirado → `MetaAPIError.is_token_expired = True`
- Falha no Facebook não cancela publicação no Instagram (try/except isolado)
- `collect_metrics` usa `apply_async(countdown=86400)` — não usa Celery Beat
- Métricas salvas em `publish_result["metrics"]` — sem campo separado no banco
- `build_full_caption()` junta caption + CTA + hashtags com `\n\n` entre partes
- `_get_request_with_client` atualizado: agora retorna `design_result`, `meta_access_token`, `instagram_business_id`, `facebook_page_id`
