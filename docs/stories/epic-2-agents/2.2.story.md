## Descrição

Como AutoPost, preciso que o Agente Copywriter use Claude Sonnet para gerar legenda, hashtags e CTA adaptados ao perfil de marca do cliente — substituindo o stub atual em `generate_copy`.

---

## Critérios de Aceitação

- [ ] AC1: Task `generate_copy` chama Claude Sonnet com `analysis_result` + `brand_profile` e retorna copy estruturado
- [ ] AC2: `copy_result` contém: `caption` (≤2200 chars), `hashtags` (lista), `cta`, `suggested_time` (horário sugerido HH:MM)
- [ ] AC3: Caption nunca inventa dados — prompt instrui Claude a usar apenas informações presentes na análise
- [ ] AC4: Tom de voz adaptado ao segmento do cliente (`brand_profile.segment`, `brand_profile.tone`)
- [ ] AC5: Hashtags incluem: nicho do segmento + localização (se disponível) + hashtags gerais Instagram
- [ ] AC6: Claude Sonnet — modelo `claude-sonnet-4-6` para qualidade máxima de copy
- [ ] AC7: Testes com mock da API Anthropic
